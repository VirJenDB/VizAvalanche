# ViralScope — Interactive Viral Sequence Explorer

An interactive choropleth map dashboard built on VirJenDB data.

**Full project:** https://github.com/Hziraksaz/ViralScope

## Features
- World map with 138 countries colored by sequence count (13M records)
- Search across viral records with family/host/country filters
- Country comparison bar chart
- Database statistics sidebar
- Snapshot system to save/load dashboard states

## Tech Stack
React 19 · TypeScript · TanStack Query · react-simple-maps · Recharts · shadcn/ui

## Run locally
bun install
bun run dev
